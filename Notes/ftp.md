## simple ftp cmd
- $> ftp [ip]
  - connects to the remote ftp and gives cmdline
- ftp$> ls [dir]
  - lists all the files in the ftp dir
- ftp$> get [file]
  - gets a file from the server
- ftp$> put [file]
  - put your own file onto the server (**This could be shell code**)
